YouTube Data API v3 키 (선택)

API 키 없이도 사용 가능합니다. YouTube RSS 피드로 최신 영상 목록을 불러올 수 있습니다.

운영자 PIN 변경

메인 페이지 로고를 3번 클릭하면 PIN을 입력해 운영자 모드로 전환됩니다.
기본값은 0000입니다. 변경하려면 새 PIN을 입력하고 저장하세요.

GitHub Pages 배포 설정

설정하면 "이번 주 발행" 버튼 클릭 한 번으로 사이트가 자동 업데이트됩니다.
설정하지 않으면 JSON 파일 다운로드 → 수동 업로드 방식으로 작동합니다.

GitHub Personal Access Token 발급 방법
GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
→ Generate new token → repo 권한 체크 → 생성

사용 방법

매주 콘텐츠 업데이트 과정 1. 왼쪽 소스 목록에서 채널을 선택합니다.
2. 피드에서 이번 주 다룰 영상/글을 선택합니다.
3. 한 줄 요약과 30~50줄 상세 설명을 작성합니다.
4. "이번 주 소식에 추가" 버튼을 클릭합니다.
5. 20개 내외 추가 후 하단 "이번 주 발행" 버튼을 클릭합니다.
6. GitHub 설정이 되어 있으면 즉시 반영, 없으면 JSON 다운로드 후 data/ 폴더에 교체합니다.

📌 피드가 안 불러와질 때
CORS 제한으로 인해 일부 피드는 직접 입력만 가능합니다.
"직접 입력하기" 버튼을 이용해 원소스 URL과 내용을 직접 입력하세요.
